# 🤠 Maverick AI

## ✨ Key Features

| Feature | Description |
| :--- | :--- |
| **🤖 Multi-Model Chat** | Converse with various AI models using a clean interface. |

- **Backend:** react +javascript+ gemini API
- **Frontend:** Streamlit
- **AI:** google gemini API
